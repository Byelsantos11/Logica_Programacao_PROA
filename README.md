# Lógica de Programação - PROA

Este projeto consiste em diversos programas desenvolvidos durante meus estudos de **Lógica de Programação no PROA**.  

Os exercícios começaram sendo feitos em **Portugol**, para aprender os conceitos básicos de programação, e depois foram **migrados para Kotlin**, aplicando os mesmos conceitos em uma linguagem de programação real.

## 📚 O que foi aprendido

Durante o desenvolvimento desses exercícios foram praticados diversos conceitos importantes, como:

- Lógica de programação
- Estrutura de código
- Estruturas condicionais (if / when)
- Estruturas de repetição (for / while)
- Entrada e saída de dados
- Organização de código
- Documentação de código
- Debugação (encontrar e corrigir erros)
- Programação em par (XP) com colegas
- Resolução de problemas com algoritmos
- Conversão de algoritmos de Portugol para Kotlin

## 🛠 Tecnologias utilizadas

- Portugol
- Kotlin
- IntelliJ IDEA
- Git
- GitHub

## 📂 Objetivo do projeto

O objetivo deste repositório é **registrar minha evolução no aprendizado de programação**, mostrando exercícios e práticas feitas durante o curso.
